① レース結果CSV（計測システム出力）

推奨ファイル名形式

R{レース番号3桁}_{計測ポイント}.csv
例意味投入先フォルダ
R001_500m.csvレース1 500m計測race_csv/500m/
R001_1000m.csvレース1 1000m計測race_csv/1000m/
R012_500m.csvレース12 500m計測race_csv/500m/
R012_1000m.csvレース12 1000m計測race_csv/1000m/

※ RowingTimerWebが出力する旧形式 YYYYMMDD_HHMMSS_R001_500m.csv も自動認識されます。

注意事項

ルール理由
500m・1000m の両方が揃ってからサイトに反映される片方だけでは不完全なため自動保留
レース番号はスケジュールのrace_noと一致させること番号が違うとデータが紐付かない
同じファイルを再投入すると上書き更新されるタイム修正時に使用

OK / NG 例

ファイル名判定理由

race1_500.csvNGRで始まらない・3桁でない

② スケジュールCSV（大会前に1回登録）

ファイル名

schedule.csv　（固定）

※ ファイル名は変更しないこと。Google Drive の master/ フォルダに投入する。

列の説明

列名必須内容・形式例
race_no必須レース番号（計測システムと一致させること）1, 2, 3...
event_code必須種目コードM_1X, W_2X, X_4+
event_name必須種目名（サイトに表示される名称）男子シングルスカル
category必須M（男子）/ W（女子）/ X（混成）M
age_group必須年齢区分 ※下記参照G, H, J
round必須FA（決勝A）/ FB（決勝B）/ SF（準決勝）/ H（予選）/ RK（順位決定）/ R（敗者復活）FA
date必須開催日（YYYY-MM-DD形式）2026-09-12
time必須発艇時刻（HH:MM形式）09:00

年齢区分（age_group）一覧

コード区分
GMasters G（70歳以上）
HMasters H（75歳以上）
A〜FMasters A〜F（各年齢区分）
Jジュニア（18歳未満）
Oオープン

③ エントリーCSV（大会前に1回登録）

ファイル名

entries.csv　（固定）

※ ファイル名は変更しないこと。Google Drive の master/ フォルダに投入する。

列の説明

列名必須内容・形式例
race_no必須スケジュールのrace_noと一致させること1, 2, 3...
lane必須レーン番号1, 2, 3, 4...
crew_name必須クルー名。ペア以上は「選手A / 選手B」形式（スラッシュ前後に半角スペース）山田太郎 / 鈴木次郎
affiliation必須所属（クラブ名・学校名等）秋田ボートクラブ
