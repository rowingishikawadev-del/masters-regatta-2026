# サンプルCSVファイル

このフォルダのCSVはGoogleDriveのmaster/フォルダにアップロードして使用します。

## ファイル一覧

| ファイル名 | 用途 | アップ先 |
|---|---|---|
| `tournament.csv` | 大会基本情報 | Drive: `master/tournament.csv` |
| `schedule_sample.csv` | スケジュール・種目一覧（サンプル） | Drive: `master/schedule.csv` にリネームして使用 |
| `entries_sample.csv` | エントリー情報（サンプル） | Drive: `master/entries.csv` にリネームして使用 |

## schedule.csv のカラム説明

| カラム | 例 | 説明 |
|---|---|---|
| race_no | 1 | レース番号（1から連番） |
| event_code | M1X | 種別コード（半角英数） |
| event_name | 男子シングルスカル | 種目名 |
| category | M / W / Mix | 性別区分 |
| age_group | G / DEF / JKLMN | 年齢カテゴリー。**複数カテゴリー合同レースは連続して記入**（例: DEF） |
| round | FA | ラウンド（FA=決勝A, FB=決勝B, H=ヒートなど） |
| date | 2026-05-23 | 開催日（YYYY-MM-DD） |
| time | 07:00 | 発艇時刻（HH:MM） |
| course_length | （空欄=1000m） | 500m種目のみ `500` と記入。空欄は大会デフォルト（1000m） |

## entries.csv のカラム説明

| カラム | 例 | 説明 |
|---|---|---|
| race_no | 1 | レース番号（scheduleと一致すること） |
| lane | 1 | レーン番号 |
| crew_name | 田中 太郎 | 選手名またはクルー名 |
| affiliation | 東京ローイングクラブ | 所属団体名 |
| **category** | **D** | **年齢カテゴリーコード（A〜N）** |

### categoryフィールドの記入ルール

- **複数カテゴリー合同レース**（age_groupが2文字以上、例: `DEF`）：**必須記入**
  - そのクルーが属するカテゴリーを1文字で記入（例: `D`, `E`, `F`）
  - これによりカテゴリー内順位が画面に表示されます
- **単一カテゴリーレース**（age_groupが1文字、例: `G`）：空欄でOK
