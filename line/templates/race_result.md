# レース結果自動通知テンプレート

## 結果通知（GAS自動送信）

```
【レース結果】{大会名}

▼ {レース番号} {種目名}

🥇 1位 {チーム名}　{タイム}
🥈 2位 {チーム名}　{タイム}
🥉 3位 {チーム名}　{タイム}

詳細はこちら
{速報サイトURL}
```

---

## 変数定義（GAS実装時の参考）

| 変数 | 内容 | 例 |
|------|------|----|
| `{大会名}` | 大会の正式名称 | 全日本マスターズレガッタ 2026 |
| `{レース番号}` | レース番号 | R-12 |
| `{種目名}` | 種目名 | M1X マスターズA |
| `{チーム名}` | チーム・団体名 | 石川ボートクラブ |
| `{タイム}` | 結果タイム | 4:23.5 |
| `{速報サイトURL}` | 詳細ページURL | https://masters-regatta-2026.pages.dev |

---

## 登録完了通知（Webhook応答）

```
【登録完了】

以下の内容で結果通知を登録しました。

チーム：{登録団体名}
レース：{登録レース番号}

結果が確定次第、このLINEにお知らせします。

登録解除する場合は「解除」と送信してください。
```
